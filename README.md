# 🧩 Autism Spectrum Disorder (ASD) Dataset

## 📌 About Dataset

Autism Spectrum Disorder (ASD) is a neurodevelopmental condition associated with significant healthcare costs. Early diagnosis can help reduce these costs, but current diagnostic procedures are often lengthy and inefficient.

To address this, this dataset has been developed to support time-efficient and accessible ASD screening methods, especially for adults. It combines behavioral and personal attributes to improve classification accuracy and identify important autistic traits.

---

## 📊 Dataset Characteristics

- **Data Type:** Multivariate  
- **Attribute Types:** Nominal (categorical), Binary, Continuous  
- **Task:** Classification  
- **Domain:** Medical, Health, Social Science  
- **Format Type:** Non-Matrix  
- **Number of Instances:** 704  
- **Number of Attributes:** 21  

---

## 🧩 Attribute Information

| Attribute | Type | Description |
|----------|------|------------|
| Age | Continuous | Age in years |
| Gender | Categorical | Male or Female |
| Ethnicity | Categorical | Common ethnicities in text format |
| Born with jaundice | Binary | Yes or No |
| Family member with PDD | Binary | Yes or No |
| Who is completing the test | Categorical | Parent, self, caregiver, clinician, etc. |
| Country of residence | Categorical | List of countries |
| Used the screening app before | Binary | Yes or No |
| Screening Method Type | Integer (0–3) | 0 = toddler, 1 = child, 2 = adolescent, 3 = adult |
| Question 1–10 Answers | Binary | Responses (0 or 1) to AQ-10 screening questions |
| Screening Score | Integer | Final score computed automatically |

---

## 📌 Key Purpose

- Enable early and efficient ASD screening  
- Reduce time and cost of diagnosis  
- Improve classification using behavioral and personal data  
- Identify key traits associated with ASD
